# Cloud Infrastructure Automation Platform

## 📌 Description
A web-based DevOps platform that automates AWS application deployment using Ansible, Docker, Kubernetes, and GitHub Actions.

## 🚀 Technologies
- React.js
- Node.js
- Express.js
- MongoDB
- AWS EC2
- Ansible
- Docker
- Kubernetes
- GitHub Actions
